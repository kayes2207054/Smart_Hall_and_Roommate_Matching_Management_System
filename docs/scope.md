# First 10 Percent Scope

This repository now has the first database foundation in place.

## Delivered

- Core Oracle tables for users, halls, rooms, seats, booking requests, and booking history
- A starter roommate matching package with a scoring function
- Initial booking workflow helpers for approvals and audit history
- A seat availability view for reporting

## Not yet implemented

- Login UI or API layer
- Advanced search/filter/sort queries
- Full analytics reports and dashboards
- Seed data and test fixtures
